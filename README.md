# SESC Tourism

## Описание
Приложение для просмотра и бронирования туристических экскурсий. Frontend — React SPA, backend — ASP.NET Core Minimal API с базой данных MS SQL Server.

## Запуск frontend
```bash
cd Front
npm install
npm run dev
```

## Запуск backend
```bash
cd Backend/sesc.API
dotnet run
```
